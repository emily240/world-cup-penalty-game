# World Cup Penalty: Spain vs Argentina

An original, responsive HTML5 Canvas penalty-kick game. All game art is drawn in code, while music and sound effects are synthesized with the Web Audio API.

## Run locally

Requires Node.js 22.13 or newer.

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal. Use a mouse on desktop or touch on mobile. Landscape orientation is recommended on phones.

## Controls

- Drag the ball before a shot to choose its position.
- Press near the player, drag backward to charge, move sideways to aim, and release to shoot.
- Longer run-ups create faster shots; excessive power reduces accuracy.

## Verify
